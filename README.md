# Power BI Dashboards

Three operational dashboards built as portfolio evidence for data analytics and business intelligence work. Each dashboard is built in Power BI Desktop with a documented data source and DAX measures.

## Dashboards

| # | Dashboard | Domain | Key Metrics |
|---|---|---|---|
| 1 | [Restaurant Operations](./01-restaurant-ops/) | F&B / Hospitality | Covers, COGS%, Labor%, Table Turns, Revenue per Cover |
| 2 | [Marketing Funnel](./02-marketing-funnel/) | Revenue Ops / Marketing | Pipeline stages, Conversion rates, CAC, MRR |
| 3 | [Stellapura Ops](./03-stellapura-ops/) | SaaS / Edge Computing | Payroll runs, Agent commissions, Holdback trends |

## How to open

1. Download the `.pbix` file from the dashboard folder
2. Open in [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free)
3. Data is embedded — no external connections needed

## Stack

`Power BI Desktop` `DAX` `Power Query (M)` `Looker Studio` `Synthetic + real data`
