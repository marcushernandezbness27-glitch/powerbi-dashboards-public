# Dashboard 2 — Marketing Funnel

Revenue pipeline dashboard tracking lead flow from source through conversion, with CAC, MRR, and win rate analysis across channels and industries.

## Pages

| Page | Purpose |
|---|---|
| Pipeline Overview | Headline KPIs + leads by source + MRR by industry + won vs lost by month |
| Funnel Drop-off | Stage drop-off funnel + win rate and CAC by source + avg days in funnel by stage |
| Cohort Trends | Monthly new leads, cumulative MRR, win rate trend, new MRR by month |

## Key metrics

| Metric | Definition |
|---|---|
| **Win Rate** | Won leads / Total leads |
| **CAC** | Customer Acquisition Cost per won deal |
| **MRR** | Monthly Recurring Revenue from won deals |
| **Avg Deal MRR** | Total MRR / Total won deals |
| **Avg Days in Funnel** | Average time from lead entry to close |

## DAX measures

```dax
Total Leads       = COUNT(funnel_leads[lead_id])
Total Won         = SUM(funnel_leads[is_won])
Total Lost        = SUM(funnel_leads[is_lost])
Win Rate          = DIVIDE(SUM(funnel_leads[is_won]), COUNT(funnel_leads[lead_id]))
Avg CAC           = AVERAGE(funnel_leads[cac])
Total MRR         = SUM(funnel_leads[mrr])
Avg Deal MRR      = DIVIDE(SUM(funnel_leads[mrr]), SUM(funnel_leads[is_won]))
```

## Data source

Two CSV files generated by `data/generate_funnel_data.py`:

- `funnel_leads.csv` — one row per lead with source, industry, stage, CAC, MRR, and outcome flags
- `funnel_monthly.csv` — pre-aggregated monthly totals for cohort trend charts

Sources: Organic Search, Paid Search, Social Media, Referral, Email Campaign  
Industries: SaaS, Healthcare, E-commerce, Finance, Agency

To regenerate:
```powershell
uv run data/generate_funnel_data.py
```
